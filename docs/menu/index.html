menu(功能栏)

2016-01-14

作者：kevinhehe
版本：v.1
依赖：mocha.css

点击运行Demo

样式1-有标题无图标及贴底 .mocha-menu-scheme1

  • 图标游戏攻略
  • 图标签到领礼
  • 图标游戏论坛

样式2-有标题及图标 .mocha-menu-scheme2

  • 图标游戏攻略
  • 图标签到领礼
  • 图标游戏论坛

样式3-有标题及图标上下结构 .mocha-menu-scheme3

  • 图标精选
  • 图标攻略
  • 图标个人
  • 图标勋章

修改模块皮肤CSS

模块的默认皮肤类.mocha-menu-skin1，您可以在您项目的page.css里修改.mocha-menu-skin1皮肤类，或者新增.mocha-menu-skin2。
