2016-03-16

作者：sonichuang
版本：v.1
依赖：mocha.css

点击运行Demo

所有文本标签

集成于mocha.css的base样式中，无需增加任何 class

h1–标题：32px

h2–栏目：30px

h3–内文：28px

h4–内文：26px

h5–副内文：24px
h6–辅助文字：20px

div,p–默认字体大小16px
